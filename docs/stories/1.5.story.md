# Story 1.5 — Responsividade Mobile

**Epic:** 1.0 — Tabela de Desempenho: Produção & Segurança
**Status:** Done
**Complexidade:** M (Medium)
**Agente:** @dev

## Objetivo
Tornar a aplicação usável em dispositivos mobile (375px+), com foco especial no Dashboard (tabela de 13 semanas) e AdminPanel.

## Acceptance Criteria

### Dashboard
- [x] AC1: Em telas < 768px, tabela de 13 semanas substituída por cards empilhados por usuário
- [x] AC2: Cada card mobile mostra: avatar, nome, role, status das métricas (farol colorido), valor atual
- [x] AC3: Em telas >= 768px, tabela tradicional mantida (comportamento atual)
- [x] AC4: Header do Dashboard responsivo — elementos empilham em mobile
- [x] AC5: Seção "Alinhamentos Recentes" scrollável horizontalmente em mobile
- [x] AC6: Seção "Destaques da Semana" empilha verticalmente em mobile

### AdminPanel
- [x] AC7: Em mobile, sidebar de lista de usuários colapsa em drawer (overlay)
- [x] AC8: Botão de toggle para abrir/fechar drawer
- [x] AC9: Formulário de edição ocupa tela inteira em mobile

### OneOnOne
- [x] AC10: Grid de métricas empilha verticalmente em mobile
- [x] AC11: Formulário de feedback responsivo com textareas em full-width

### Geral
- [x] AC12: Nenhum overflow horizontal em qualquer tela em 375px
- [x] AC13: Touch targets mínimos de 44px em botões e links

## Arquivos Impactados
- `components/Dashboard.tsx` — layout alternativo mobile
- `components/AdminPanel.tsx` — drawer mobile para sidebar
- `components/OneOnOne.tsx` — ajustes de grid
- Possivelmente componente utilitário `MobileCard.tsx` para cards do dashboard

## Notas Técnicas
- Usar classes responsive do Tailwind: `hidden md:block`, `block md:hidden`, `md:grid-cols-X`
- Login.tsx e CheckIn.tsx já são responsivos (max-w-md/max-w-2xl)
- Testar em viewport 375x812 (iPhone SE) como baseline

## Dependências
- Story 1.4 (Tailwind build-time para classes responsive funcionarem corretamente)

## Definition of Done
- Dashboard usável em 375px sem overflow horizontal
- AdminPanel navegável em mobile
- Todos os botões com touch target >= 44px
