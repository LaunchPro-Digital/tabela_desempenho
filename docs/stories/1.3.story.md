# Story 1.3 — Row Level Security (RLS)

**Epic:** 1.0 — Tabela de Desempenho: Produção & Segurança
**Status:** Done (implementada na Story 1.2 — migration_cloud.sql linhas 62-110)
**Complexidade:** M (Medium)
**Agente:** @dev

## Objetivo
Ativar Row Level Security em todas as tabelas do schema `desempenho` para que CONTRIBUTORs só acessem seus próprios dados e PARTNERs tenham acesso total.
